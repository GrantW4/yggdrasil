## What is Yggdrasil?

In Norse mythology, Yggdrasil is the immense sacred tree that connects all nine worlds Рђћ its roots reach into the earth, its branches touch the heavens, and every living thing depends on it. It is the axis of existence.

This curriculum is built on the same idea. **Real AI mastery isn't a ladder Рђћ it's a tree.** The roots are mathematics and programming. The trunk is classical machine learning. The branches are deep learning, LLMs, and agents. The canopy is production systems engineering. The whole structure is interconnected, load-bearing, and alive.

Yggdrasil takes you from absolute zero to the skills required to design, build, train, and ship AI systems at scale Рђћ the kind of work happening at Anthropic, OpenAI, Google DeepMind, and the most ambitious AI startups in the world.

> "An expert is a person who has made all the mistakes that can be made in a very narrow field." Рђћ Niels Bohr

This curriculum is built so you make those mistakes in a controlled environment, on real problems, with real tools Рђћ before you're doing it in production.

## The Curriculum at a Glance

Yggdrasil is organized into **7 Phases**, each building on the last.

```
Phase 0: Roots        Рђћ Mathematics, Python, CS Fundamentals
Phase 1: Trunk        Рђћ Classical Machine Learning  
Phase 2: Branches     Рђћ Deep Learning & Neural Networks
Phase 3: The LLM Era  Рђћ Transformers, LLMs, Fine-tuning, RAG
Phase 4: Canopy       Рђћ AI Systems Engineering & MLOps
Phase 5: The Realms   Рђћ Agents, Multimodal AI, AI Safety
Phase 6: Elite        Рђћ Research Skills & Capstone Projects
```

Each phase contains **modules**, each module contains **lessons**, and each lesson contains:
- Clear **learning objectives**
- Curated **reading material** (papers, docs, blog posts)
- **Hands-on exercises** you build and run yourself
- A **knowledge check quiz**
- A **project** that ties concepts together

No video lectures you passively absorb. No multiple-choice tests that prove nothing. **You learn by building.**

## How to Use This Curriculum

### 1. Work Linearly (Recommended for Beginners)
Start at Phase 0, Module 1, Lesson 1. Don't skip. The curriculum is carefully sequenced so each concept builds on the last.

### 2. Jump In (For Those with Background)
Already know Python and some ML? Take the [placement quiz](curriculum/PLACEMENT.md) to find your entry point. Most people with a CS background start at Phase 1 or Phase 2.

### 3. Use It as a Reference
Already working in AI? Use Yggdrasil as a structured reference. Every module has standalone value. Browse by topic.

### The Daily Practice

The single best way to use Yggdrasil:
```
1. Read the lesson objectives Рђћ know what you're trying to understand
2. Work through the reading material actively (take notes, draw diagrams)
3. Complete the exercises Рђћ do not skip these
4. Take the quiz Рђћ if you fail, re-read and retry
5. Build the project Рђћ this is where learning becomes skill
6. Commit your work to a public repo Рђћ your portfolio matters
```
