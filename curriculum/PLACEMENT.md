# Placement Guide 🌳

Use this guide to find where you should start in
The Yggdrasil Project based on your current experience.

## Complete Beginner
Start at Phase 0: Roots. No prior experience needed.

## Some Python Experience
Start at Phase 1: Trunk. You know basic Python but
haven't done ML yet.

## ML Background
Start at Phase 2: First Branches. You know sklearn
and basic neural networks.

## Deep Learning Experience
Start at Phase 3: LLMs. You can train models and
understand backprop.

## LLM Experience
Start at Phase 4: Canopy. You've built RAG systems
and worked with LLMs in production.

## Advanced
Start at Phase 5: Realms or Phase 6: Elite.
